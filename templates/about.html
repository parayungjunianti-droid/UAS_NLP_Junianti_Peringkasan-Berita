Sistem Peringkasan Berita Bahasa Indonesia

Proyek UAS Natural Language Processing

Aplikasi web modern berbasis NLP untuk menghasilkan ringkasan berita secara otomatis dalam Bahasa Indonesia menggunakan pendekatan Transformer Abstraktif.

Spesifikasi Teknologi

Versi Python Python 3.10+
Web Framework Flask 3.0
NLP Model csebuetnlp/mT5_multilingual_XLSum
AI Library Hugging Face & PyTorch
Data Processing Pandas & NumPy
UI Design Bootstrap 5 + Custom SaaS CSS

Fitur Aplikasi

  • Peringkasan Berita Abstraktif:
    Generasi teks ringkasan berbasis makna secara otomatis.
  • Validasi Input Teks:
    Mengecek input kosong dan batas panjang minimal 50 karakter.
  • Export Dokumen Ringkasan:
    Dukungan Copy Teks, Unduh file TXT, dan file PDF.
  • Evaluasi Metrik NLP:
    Kalkulasi ROUGE-1, ROUGE-2, ROUGE-L, BLEU, dan grafik Chart.js.
  • Desain Responsive SaaS:
    Optimasi tampilan desktop, tablet, dan mobile.
