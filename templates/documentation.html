1. Pengertian Text Summarization

Text Summarization (Peringkasan Teks) adalah salah satu cabang utama dalam Natural Language Processing (NLP) yang bertujuan untuk mereduksi dokumen teks panjang menjadi ringkasan yang lebih singkat tanpa menghilangkan substansi dan informasi utama dokumen asli.

Peringkasan otomatis sangat krusial dalam domain berita digital untuk membantu pembaca memahami poin penting secara cepat dan efisien di tengah melimpahnya informasi (information overload).

2. Metode Abstraktif

Pendekatan peringkasan teks terbagi menjadi dua kategori utama:

  • Extractive Summarization: Mengambil kalimat-kalimat utuh secara langsung dari teks sumber berdasarkan bobot statistik tanpa mengubah susunan kata.
  • Abstractive Summarization: Memahami konteks semantik teks sumber dan merumuskan kalimat-kalimat baru yang tidak secara verbatim muncul di dokumen asli. Metode ini meniru cara berpikir manusia dalam merangkum.

Aplikasi ini memanfaatkan metode Abstractive Summarization sehingga dapat menghasilkan ringkasan yang lebih alami, fleksibel, dan koheren.

3. Cara Kerja Transformer

Arsitektur Transformer berbasis mekanika Self-Attention Mechanism memungkinkan model memproses hubungan antar kata secara eksplisit dalam urutan paralel.

Model sequence-to-sequence seperti mT5 (Multilingual T5) dan BART dilatih menggunakan mekanisme Encoder-Decoder:

  • Encoder: Memetakan urutan token berita Bahasa Indonesia ke dalam ruang vektor konteks tersembunyi (hidden representation).
  • Decoder: Menggenerasi token demi token ringkasan baru secara terarah berdasarkan representasi dari Encoder.

4. Tahapan Preprocessing

Sebelum dimasukkan ke dalam model Transformer, teks berita melalui tahapan pembersihan:

  1. Case Folding: Mengubah seluruh huruf menjadi huruf kecil (lowercase).
  2. Normalisasi Whitespace: Menghapus karakter spasi berlebih, newline, dan tabulasi.
  3. Pembersihan Karakter: Menghapus simbol khusus yang tidak relevan.
  4. Tokenisasi Subword: Memecah kata menjadi unit subword/token untuk dimasukkan ke dalam model.

5. Dataset yang Digunakan

Sistem ini dievaluasi menggunakan dataset berita Bahasa Indonesia yang tersimpan pada berkas dataset/news_dataset.csv. Dataset mencakup artikel berbagai bidang seperti teknologi, ekonomi, infrastruktur, dan lingkungan hidup beserta ringkasan acuan manusia (human reference summary).

6. Cara Menjalankan Aplikasi

Ikuti langkah berikut untuk menjalankan aplikasi secara lokal:

# 1. Install dependensi
pip install -r requirements.txt

# 2. Jalankan server Flask
python app.py

Akses aplikasi melalui peramban web di alamat http://localhost:5000.
